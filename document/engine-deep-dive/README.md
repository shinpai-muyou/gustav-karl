# Gustav Karl 渲染系统深度文档（中文）

本目录提供三份互补文档：

1. `01-架构总览.md`：系统架构、模块边界、资源关系。
2. `02-算法与Argo说明.md`：核心算法（shader 预处理、测地线追踪、Bloom、状态打包）。
3. `03-执行逻辑与时序.md`：应用启动、每帧执行顺序、Pass 依赖与数据流。

图表说明：

1. 三份文档均已内置 Mermaid 图（架构图、时序图、算法流程图、数据流图）。
2. 若你的 Markdown 预览器不支持 Mermaid，请用支持 Mermaid 的预览插件或平台查看。

阅读建议：

1. 先看 `01-架构总览.md` 建立全局图。
2. 再看 `03-执行逻辑与时序.md` 理解代码如何跑起来。
3. 最后看 `02-算法与Argo说明.md` 理解“为什么这样算”。
